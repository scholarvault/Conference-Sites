Submission Guidelines

How to Submit

Paper Requirements

  • → Original, unpublished work not under review elsewhere
  • → Manuscript: 4–8 pages (research) / 2–4 pages (short comm.)
  • → Format: PDF or DOCX using provided template
  • → Language: English only
  • → Include: abstract (250–500 words), keywords (5–8), references
  • → Double-blind review — remove all author identifiers

Review Process

  • ✓ Double-blind peer review (2–3 reviewers per paper)
  • ✓ Reviews: originality, methodology, significance, clarity
  • ✓ Accepted papers published in ISBN proceedings
  • ✓ DOI assigned to all accepted papers
  • ✓ ScholarVault anti-predatory review guaranteed
  • ✓ Plagiarism check via iThenticate
